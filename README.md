# Vault Obsidian - IHM-1

Ce dossier est un **vault Obsidian** contenant du contenu pour le cours IHM-1.

## ⚙️ Prérequis

- [Obsidian](https://obsidian.md/) doit être installé sur votre machine

## 📂 Comment utiliser ce vault

1. **Télécharger et installer Obsidian** (si ce n'est pas déjà fait)
   - Rendez-vous sur [obsidian.md](https://obsidian.md/)
   - Téléchargez et installez la version correspondant à votre système d'exploitation

2. **Ouvrir le vault dans Obsidian**
   - Lancez Obsidian
   - Cliquez sur "Ouvrir un dossier comme vault" (ou "Open folder as vault")
   - Naviguez jusqu'à ce dossier (`IHM-1`)
   - Sélectionnez-le pour l'ouvrir

3. **Explorer le contenu**
   - Une fois le vault ouvert, vous verrez tous les fichiers et notes
   - Naviguez entre les documents via les liens ou la barre latérale

## 📝 Contenu du vault

- **Index.md** - Table des matières et navigation globale
- **Introduction.md** - Introduction au cours
- **Flux d'un système D'Information.canvas** - Diagramme visuels

## 💡 Tips Obsidian

- Utilisez les liens `[[]]` pour créer des connexions entre notes
- Le graph view visualise les connexions entre vos notes
- Vous pouvez utiliser des plugins pour enrichir votre expérience
